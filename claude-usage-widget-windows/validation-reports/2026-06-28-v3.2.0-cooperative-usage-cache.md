### Phase 4: Code Quality
- New module `src/usage_cache.py` is Qt-free (imports `config` + `oauth` only). Clear separation: `oauth` owns credentials/API, `usage_cache` owns cross-process coordination.
- A single `next_attempt_at` gate unifies freshness and backoff, which let `run_tui`'s per-process exponential backoff (`MAX_INTERVAL`/`MAX_BACKOFF_LEVEL`, the local `_age`) be **removed** — net simplification of the loop.
