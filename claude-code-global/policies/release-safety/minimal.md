# Release Safety: Minimal

> **Policy module**: Activated via `## Selected Policies` in project `.claude/CLAUDE.md`.
> For prototypes, experiments, and throwaway projects where release safety overhead is not justified.

---

## Requirements

- **Acknowledge risk**: Understand that changes may not be easily reversible
- **No formal review required**: Skip Phase 5.5 checklists entirely
- **Git revert is the rollback plan**: No additional documentation needed
