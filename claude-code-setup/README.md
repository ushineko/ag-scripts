# Claude Code Setup

A script to install and verify `@anthropic-ai/claude-code`, the official CLI tool for Claude.

## Version
0.1.0

## Description
This project contains scripts to install and uninstall the Claude Code CLI tool. It handles:
- Checking for Node.js and npm dependencies.
- Installing the package globally via npm.
- Verifying the installation.

## Usage
### Install
Run the installation script:
```bash
./install.sh
```

### Uninstall
Run the uninstallation script:
```bash
./uninstall.sh
```

## Requirements
- Node.js and npm (The installer will error if not found).
